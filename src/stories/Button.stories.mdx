# Buttons

To trigger an operation

In Artemis UI we have 7 variants of Buttons

- Default
- Primary
- Primary Light
- Danger
- Danger Light
- Success
- Success Light

`Props` are listed at the end of this page.

[View source](https://github.com/iamrishupatel/artemis-ui/tree/main/src/components/Button)

## Imports

```js
import { Button } from "artemis-ui";
```

### Button Sizes

Button comes in three sizes.
By default they are of **_medium_** size

Use the `size` prop to change the size of the `Button`.
You can set the value to `sm`, `md`, `lg`

<Canvas withSource="open">
  <Story
    name="sizes"
    parameters={{
      docs: {
        source: {
          code: `
<Button size="sm">Small</Button>
<Button size="md">Medium</Button>
<Button size="lg">Large</Button>
`,
        },
      },
    }}
  >
    <div
      style={{
        display: "flex",
        gap: "1rem",
        alignItems: "center",
        flexWrap: "wrap",

      <Button size="sm">Small</Button>
      <Button size="md">Medium</Button>
      <Button size="lg">Large</Button>
    </div>

### Button Variants

Use the variant prop to change the visual style of the `Button`. You can set the value to `primary`, `success`, `danger`, `primary-light`,`success-light`, or `danger-light`.

<Canvas withSource="open">
  <Story
    name="variants"
    parameters={{
      docs: {
        source: {
          code: `
<Button variant="primary">Primary</Button>
<Button variant="success">Success</Button>
<Button variant="danger">Danger</Button>
<Button variant="primary-light">Primary light</Button>
<Button variant="success-light">Success light</Button>
<Button variant="danger-light">Danger light</Button>
`,
        },
      },
    }}
  >
    <div
      style={{
        display: "flex",
        gap: "1rem",
        alignItems: "center",
        flexWrap: "wrap",
      }}
    >
      <Button variant="primary">Primary</Button>
      <Button variant="success">Success</Button>
      <Button variant="danger">Danger</Button>
      <Button variant="primary-light">Primary light</Button>
      <Button variant="success-light">Success light</Button>
      <Button variant="danger-light">Danger light</Button>
    </div>
  </Story>
</Canvas>

| State                | Description                                                                                                                                                                                    |
| -------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Primary button       | Use a Primary button to inform about the importance of an action. This is the strongest call to action. No more than one primary button should appear on a screen. Examples: Save, Finish, ... |
| Success button       | Use a Success button for reassurance. Use it preferably to indicate a validation, a success, an end of a workflow, ... Examples: Confirm, Save, ...                                            |
| Danger button        | Use a Danger button to indicate a destructive action. Example: Delete                                                                                                                          |
| Primary-light button | Use a Primary-light button to indicate a lower emphasis style to a primary action.                                                                                                             |
| Success-light button | Use a Success-light button to indicate a lower emphasis style to a successful action.                                                                                                          |
| Danger-light button  | Use a Danger-light button to indicate a lower emphasis style to a destructive action. Example: Delete this entry                                                                               |
