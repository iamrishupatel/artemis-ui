# Tootltip

Tooltips are floating labels that display additional information upon hover of an element.

- The tooltip should not be used to provide critical or urgent information.
- Tooltips should be used sparingly.
- We have a max-width of 150px

`Props` are listed at the end of this page.

[View source](https://github.com/iamrishupatel/artemis-ui/tree/main/src/components/Tooltip)

## Imports

```js
import { Tooltip } from "artemis-ui";
```

## Usage

Tooltips are used to show the full version of truncated text or is a way to display further information about an element. The tooltip will appear on top of adjacent text.

<Canvas withSource="open">
  <Story name="base">
    <div>
      <span>
        Lorem ipsum dolor sit amet consectetur adipisicing elit. Ducimus
        excepturi at sunt eum magni iste reprehenderit rem rerum dolorum sit

      </span>
      <Tooltip message="I'm a tooltip">
        <strong>Hover ME</strong>
      </Tooltip>

        Lorem ipsum dolor sit amet consectetur adipisicing elit. Ducimus

      </span>

</Canvas>

### Placements

You can change the placement of the tooltip by changing the `placement` prop. The default `placement` is top.

        Left
      </Tooltip>
      <Tooltip placement="right" message="I'm a tooltip">
