# Tabs

Tabs make it easy to switch between different views.

Best Practices

- Tabs label should be concise. One word or two maximum is preferable.

`Props` are listed at the end of this page.

[View source](https://github.com/iamrishupatel/artemis-ui/tree/main/src/components/Tabs)

## Imports

```js
import { TabGroup, Tabs, Tab, TabPanels, TabPanel } from "artemis-ui";
```

## Usage

Tabs allow to classify related content in different views within a same page.
