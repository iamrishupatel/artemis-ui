# DropDown

When there are more than a few options to choose from, you can wrap them in a `Dropdown`.
By hovering or clicking on the trigger, a dropdown menu will appear

`Props` are listed at the end of this page.

[View source](https://github.com/iamrishupatel/artemis-ui/tree/main/src/components/DropDown)

## Imports

```js
import { DropDown, DropDownItem } from "artemis-ui";
```

## On Hover

The default trigger mode is `click`, you can change it to `hover` using `trigger` prop.

<Canvas withSource="open">
  <Story
    name="hover"
    parameters={{
      docs: {
        source: {
          code: `
<DropDown trigger="hover" label="Hover Me">
  <DropDownItem>Item 1</DropDownItem>
  <DropDownItem>Item 2</DropDownItem>
  <DropDownItem>Item 3 is a bit longer</DropDownItem>
</DropDown>
`,
        },
      },
    }}
  >
    <div style={{ height: "10rem" }}>
      <DropDown trigger="hover" label="Hover Me">
        <DropDownItem>Item 1</DropDownItem>
        <DropDownItem>Item 2</DropDownItem>
        <DropDownItem>Item 3 is a bit longer</DropDownItem>
      </DropDown>
    </div>
  </Story>
</Canvas>
