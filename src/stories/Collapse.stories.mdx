# Collapse

Collapse are collapsable blocks that vertically hide a stack of content.
Clicking on the block reveals the associated content.

- Can be used to group or hide complex regions to keep the page clean.
- Use collapse for lower priority information.

`Props` are listed at the end of this page.

[View source](https://github.com/iamrishupatel/artemis-ui/tree/main/src/components/Collapse)

## Imports

```js
import { Collapse, CollapseItem } from "artemis-ui";
```

### Types

use `type` prop to change visual style of a collapse

| Type          | Description                     |
| ------------- | ------------------------------- |
| border        | add borders to collapse item    |
| margin        | add margin to collapse item.    |
| basic         | basic collapse                  |
| margin-border | collapse with margin and border |
