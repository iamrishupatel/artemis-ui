# TextArea

A basic component for getting more than one line as input

Best Practices

- Labels should be clear and concise.
- Make sure to use the placeholder feature whenever clarification are helpful on how to use the TextArea.
- Make sure to use to define the TextArea as required whenever it is absolutely mandatory.

`Props` are listed at the end of this page.

[View source](https://github.com/iamrishupatel/artemis-ui/tree/main/src/components/FormComponents)

## Imports

```js
import { TextArea } from "artemis-ui";
```

## Usage

TextAreas are always found in forms. They give users the possibility to add or edit multiline text.
