# Select

Select component to select value from options.

`Props` are listed at the end of this page.

[View source](https://github.com/iamrishupatel/artemis-ui/tree/main/src/components/FormComponents)

## Imports

```js
import { Option, Select } from "artemis-ui";
```

## Usage

<Canvas withSource="open">
  <Story name="default">
    <Select defaultValue="select">
      <Option value="select" disabled>
        Select
      </Option>
      <Option value="it1">Item 1</Option>
      <Option value="it2">Item 2</Option>
      <Option value="it3">Item 3</Option>
      <Option value="it4">Item 4</Option>
    </Select>
  </Story>
</Canvas>

## With Errors

Select can display errors. The error message is customizable and is shown below the writing area.

<Canvas withSource="open">
  <Story name="errors">
    <Select defaultValue="select" error="this is an error">
      <Option value="select" disabled>
        Select
      </Option>
      <Option value="it1">Item 1</Option>
      <Option value="it2">Item 2</Option>
      <Option value="it3">Item 3</Option>
      <Option value="it4">Item 4</Option>
    </Select>
  </Story>
</Canvas>
