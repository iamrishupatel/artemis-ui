# Badge

Badges are used to give a quick visual indication to the users.

Best practices

- Use them next to a label or a description.
- Badge component should be filled only with numbers or a single word max.

`Props` are listed at the end of this page.

[View source](https://github.com/iamrishupatel/artemis-ui/tree/main/src/components/Badge)

## Imports

```js
import { Badge } from "artemis-ui";
```

An icon can be added to badge using `icon` prop to give more impact

### Sizes

Badge comes in three sizes.
By default they are of **_medium_** size

Use the `size` prop to change the size of the `Badge`.
You can set the value to `sm`, `md`, `lg`

<Canvas withSource="open">
  <Story
    name="sizes"
    parameters={{
      docs: {
        source: {
          code: `
<Badge size="sm">small</Badge>
<Badge size="md">medium</Badge>
<Badge size="lg">large</Badge>
`,
        },
      },
    }}
  >
    <div
      style={{
        display: "flex",
        gap: "1rem",
        alignItems: "center",
        flexWrap: "wrap",
      }}
    >
      <Badge size="sm">small</Badge>
      <Badge size="md">medium</Badge>
      <Badge size="lg">large</Badge>
    </div>
  </Story>
</Canvas>
