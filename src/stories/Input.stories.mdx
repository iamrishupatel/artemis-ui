# Input

A basic component for getting the user input. Keyboard and mouse can be used for providing or changing data.

Best Practices

- Labels should be clear and concise.
- Make sure to use the placeholder feature whenever clarification are helpful on how to use the input.
- Make sure to use to define the input as required whenever it is absolutely mandatory.
- If a single line is not enough for the amount of text required, use `TextArea` component.

`Props` are listed at the end of this page.

[View source](https://github.com/iamrishupatel/artemis-ui/tree/main/src/components/FormComponents)

## Imports

```js
import { Input } from "artemis-ui";
```
